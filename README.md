
## 📝About
Opening a youtube video using a simple vbscipt program.

## 👨‍💻 Extra:
If you want to open cmd or anything else using a vbscript program the syntax is :

```sh
set x=createobject("wscript.shell")

x.run"cmd.exe"
wscript.sleep 2000
x.sendkeys"color A"
x.sendkeys"{enter}"
'x.sendkeys"exit"
'x.sendkeys"{enter}"



```
